Grave Database (CDB)
Live Application

Access the live app here: gravedatabase.com

Overview

Grave Database (CDB) is a full-stack web application designed to store, manage, and search cemetery records. The application allows users to efficiently organize and retrieve structured data such as names, dates, and burial locations.

This project demonstrates end-to-end software development, including database design, backend logic, and live cloud deployment.

---------------------------------------------------------------------------------------------------
Features

Search functionality to locate records based on user input

Structured data storage including:
First and last name,
Birth and death dates,
Cemetery name,
Section and lot number
Full CRUD operations (create, read, update, delete)
Dash app to create clean UI
Live deployment accessible through a public domain

---------------------------------------------------------------------------------------------------

Tech Stack

Backend: Python (Dash / Flask)

Database: MongoDB Atlas

Deployment: Render

Version Control: Git and GitHub
